PRIVACY_POLICY.md# Roam Privacy Policy

**Effective Date:** June 2026

## Overview

Roam is a local-first iOS app that collects and tracks the ZIP Code Areas you visit, privately on your device.

## Data Collection

### Location Data
- Your location is recorded when you enable the app
- Background location tracking uses iOS location services
- Locations are processed locally to identify ZIP Code Areas (ZCTAs)
- Location history is stored locally on your device

### Data Storage
- All location history stored locally in app's private container
- No cloud sync
- No account required
- Data persists only on your device

## Third-Party Services

Roam uses:
- **iOS Location Services** - On-device processing only
- **Census ZIP Code data** - Static reference data, not transmitted

Roam does NOT use:
- Analytics
- Crash reporting
- Cloud storage
- Ads
- Third-party tracking

## Permissions

**Location Permission** - Required for core functionality. Used only for ZCTA detection.

## Data Control

You have full control:
- Delete individual visits in the app
- Clear all history
- Disable background tracking
- Uninstall to remove all data
